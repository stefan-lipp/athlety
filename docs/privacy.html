Verantwortlicher

Angaben zum Verantwortlichen findest du im Impressum.

Datenverarbeitung in der App

Athlety erhebt keine personenbezogenen Daten. Es gibt keine Benutzerkonten, keine Registrierung und keine Anmeldung. Die App kann vollständig anonym genutzt werden.

Netzwerkverbindungen

Die App ruft öffentlich verfügbare Wettkampfdaten über die Schnittstelle der Leichtathletik-Datenverarbeitung (LADV) ab. Dabei werden ausschließlich technische Parameter (z. B. ein API-Schlüssel und optionale Filtereinstellungen) übermittelt — keine Daten, die dich als Person identifizieren. Die gesamte Kommunikation erfolgt verschlüsselt über HTTPS.

Lokale Datenspeicherung

Athlety speichert folgende Daten lokal auf deinem Gerät:

  • Einstellungen wie das gewählte Erscheinungsbild (Hell/Dunkel/System) und der zuletzt ausgewählte Landesverband
  • Von dir gemerkte Wetkämpfe (Lesezeichen) mit Name, Ort und Datum

iCloud-Synchronisation

Deine Lesezeichen werden über Apples iCloud-Infrastruktur zwischen deinen Geräten synchronisiert, sofern du iCloud auf deinem Gerät aktiviert hast. Für die Verarbeitung dieser Daten gelten die Datenschutzrichtlinien von Apple.

Kalenderzugriff

Wenn du einen Wettkampf in deinen Kalender exportierst, fragt die App um Erlaubnis, auf deinen Kalender zuzugreifen. Kalendereinträge werden ausschließlich auf deine ausdrückliche Aktion hin erstellt. Es findet kein automatischer oder hintergründiger Zugriff auf deinen Kalender statt.

Keine Analyse- oder Tracking-Dienste

Athlety verwendet keine Analyse-, Werbe- oder Tracking-Dienste. Es werden keine Nutzungsdaten, Geräteinformationen oder Absturzberichte an Dritte übermittelt.

Website

Diese Website ist eine statische Seite ohne Cookies, Analyse-Tools oder Kontaktformulare. Sie wird über GitHub Pages gehostet. Beim Aufruf der Seite können durch GitHub automatisch Server-Logdaten (z. B. IP-Adresse, Browsertyp, Zeitpunkt des Zugriffs) erhoben werden. Näheres regelt die Datenschutzerklärung von GitHub.

Deine Rechte

Nach der DSGVO hast du unter anderem das Recht auf Auskunft, Berichtigung, Löschung und Einschränkung der Verarbeitung deiner Daten sowie das Recht auf Beschwerde bei einer Aufsichtsbehörde. Da Athlety keine personenbezogenen Daten erhebt, fallen in der Regel keine Daten an, auf die sich diese Rechte beziehen.

Änderungen
